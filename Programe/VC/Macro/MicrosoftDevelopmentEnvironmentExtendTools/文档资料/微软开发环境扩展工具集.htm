    此版本是原来的《 VC 扩展工具集》的升级版，版本号：4.1 。经过了一年的开发，现在有在 VS6.0 下运行的多个版本；还有在 VS.NET 下运行的版本。

    使用本Add_In:

    一、下载本 Add_In 。然后把它释放到一个目录中（假设VC6.0安装在C盘，建议不释放到C:\Program Files\Microsoft Visual Studio\Common\MSDev98\AddIns 中，如果释放此目录，在加载时会提示不能注册，请不要理会。详见第三步。）。
    二、打开VC6.0。
    三、打开菜单中的 “工具”->“定制”。出现下面的对话框（如图一）。选择“附加项和宏文件”标签。如果你把三个文件直接释放到VC6.0默认的Add_In目录中，那么在这一步时会出现错误提示框，说一些 DLL 不能注册，不要理会，点确定。如果没有出现图一中红圈中的项（MicrosoftDevelopmentEnvironmentExtendTools)。那就点击浏览按钮，找到你所释放的文件夹，选中MicrosoftDevelopmentEnvironmentExtendTools.dll文件，再点击确定。就加了 微软开发环境扩展工具集 - VS6.0。然后再点击关闭。就可以了。

功能：

  1. 自动格式化 C 代码

  2. 用文本方式打开资源文件

  3. 添加头文件

  4. 头文件与实现文件互换

  5. 常量、宏自动提示器用法

  6. 自动完成右边的括号输入

    ;错误警告
    ErrorWarning = TRUE
