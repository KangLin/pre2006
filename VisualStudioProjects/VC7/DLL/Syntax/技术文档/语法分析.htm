语法分析

康  林  2005年5月

一、在程序块的结束行右方加注释标记，以表明某程序块的结束。

if、while、do... while、for

if(bFlag)
{
    return;
}

  • 输入完整的条件行:
    输入 if(bFlag) 自动完成:
    if(bFlag)
    {

    }//if(bFlag)
  • 输入'{':
    if(bFlag){ 自动完成:
    if(bFlag)
    {

    }//if(bFlag)
  • 输入完整行:
    if(bFlag){return;} 自动完成:
    if(bFlag)
    {
        return;
    }//if(bFlag)

条件行分为多行输入:
 输入
if(a == b
   && c == d)
自动完成:
 if(a == b
   && c == d)
 {

}//if(a == b

二、当修改 一 中的语句时，自动修改相应的大括号后的注释。（2005年06月23日）
 只考虑了在关键行修改的情况。例如：
if(a == b)改成 if(a != b),则相应大括号后的注释进行修改。
但是不在关键字行修改时，不进行相应的修改。例如：
if(a == b
    && c == d)//在这行修改时，相应的注释行不进行修改。
