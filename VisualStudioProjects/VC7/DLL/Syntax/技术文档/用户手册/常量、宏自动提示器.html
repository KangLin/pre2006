Microsoft Development Environment 扩展工具集

常量、宏自动提示器

功能:
提示常量、宏,并完成自动输入。能够记录用户自己定义的宏。并在提示器中显示。
用法：
激活字符：
要显示系统提供的默认宏，默认激活字符为“..”。
要显示用户自己定义的宏，默认激活字符为“.>”。
你可在配置文件中设置自己的激活字符。
注意：如果用户把系统提供的宏的激活字符与用户自己定义的宏的激活字符设置为相同的话，那么用户自己定义的宏自动增加到系统提供的宏中。也就是说用户自己定义的宏与系统提供的宏一起显示。当关闭解决方案时，用户自己定义的宏将被从系统中删除。

取消显示用 “ ESC ” 键。

选取项目:用Alt+光标键(Up、Down)：单行；
           Alt+（PAGEUP、PAGEDOWN）：翻页；
           Alt+（HOME、END）：至顶、至尾；
           也可以用鼠标。选中后双击，即可自动完成输入。

自动完成输入：
    当输入空格、分号“；”、大括号、小括号、中括号、问号、单引号、双引号、冒号“:”、算术运算符号（“+”、“-”、“*”、“/”、“%”、“\”）、逻辑运算符号（“&”、“|”、“^”、“~”、“!”)等时，自动完成输入。
配置文件：AutoDisplayConstOrmacro.InI
[Settings]
;显示系统提供的宏的激活键
PromptString = ..
;显示用户添加的宏的激活键
UserPromptString = .>
;当显示提示框时，按“ESC”键的作用。TRUE：清除已输入的字符。FALSE：不清除已输入的字符。
ESCFunction = TRUE
;包含要显示的内容的文件
ListFile = "List.txt"
宏例表文件：在配置文件中的 ListFile 中设置。默认为：List.txt
此文件包含系统提供的宏、常量。
文件格式：一条宏或常量占一行。
我已用工具《得到文件中的宏》从VS.NET2003中提出了VC7目录下的宏放在此文件中。在我机器上用了一天时间。因为在提取时中断了几次，所以可能没有完全提取出所有的宏。只提取了“C:\Program Files\Microsoft Visual Studio .NET 2003\Vc7\PlatformSDK\Include”下的宏 list.txt 。总共有 129309 条。由于太大，所能没放到程序文件夹中，如果你的内存够大，你可用下面的工具把它与程序文件夹中的 list.txt 合并，以便使用。如果你有兴趣的话，你可用我提供的工具重新提取“C:\Program Files\Microsoft Visual Studio .NET 2003\Vc7\”一遍。如果你是一次性的提取，请你把 List.txt 文件发给我。当然，如果你用 C#、JAVA，那你要把 list.txt 换成相应的宏列表文件。因为我没用过这两种语言，所以没有提取，不知它们中的宏定义是否与 C\C++ 中的一样，如果一样，那也可用我提供的这个工具提取。如果你有办法提取它们中的宏，请把你的程序发给我。
工具：
《得到文件中的宏》
从头文件（文件扩展名为.h，其它文件不读取）中提取宏,宏被保存在当前目录下的 List.txt 文件中。
菜单中的“工具->合并宏”，合并两文件中的宏，并去掉重复的项。

位于安装文件夹中，在“开始->所有程序->微软开发环境扩展工具集”下。
