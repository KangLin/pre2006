Microsoft Development Environment 扩展工具集

序言

随着本人从 VC6.0 升级到了 VS.NET 2003 。我觉得 VS.NET 2003 在以下几个方面有所改进：

  • 增加的查找类、函数的实现功能。方便了我们可以查看它的代码。
  • 代码编辑器能够折叠代码，便于我们浏览。
  • 其帮助文档中的 VB.NET、.NET、C/C++ 中一部分（MFC）都是汉化的。
  • 采用了可自动隐藏的工具窗口，使我们有了更大的编辑视图。
  • 提供了选卡式文档，可以方便在各打开文档之间切换。

不足之处：

  • 占用系统内存太多，程序冗杂，速度较慢。

    本人把去年写了 VC 扩展工集进行了升级。使其支持 VS.NET 2003。最终命名为《 微软开发环境扩展工具集 》。

    重新写了《常量、宏自动提示器》功能。并增加了新的功能――自动格式化 VC 代码。使 VC 能象 VB 一样能在输入时自动规格化代码。使用户在写 VC 程序时，代码能自动满足代码的《编程规范和范例》。 还有其它一些功能，详见功能中的说明。
    大家在使用时发现 BUG （可能的话分析错误原因和修改意见。）并向我报告（BUG 报告表）。并向本人提出你希望增加的功能。被采用的将免费得到此软件的正式发行版本。
欢迎各位与本人探讨。

写本程序的目的：提高编程的效率。并使代码规范化。

本人运行环境：
硬件：Inter C4 1.8G、256M DDR、Inter 845G
软件：中文版 Windows XP Professional 5.1(内部版本号 2600.xpsp1.020828-1920:Service Pack 1)
      中文版 Microsoft Office Professional Eidtion 2003
      中文版 Microsoft Visual Studio .NET 2003 版本号：7.1.3091
