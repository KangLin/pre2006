对齐注释

VisualStudioNetExtendTools.Connect.AlignComment:

功  能:对齐注释

用  法:选取要对齐注释的文本,执行此命令
效果如下:

struct UDPSOCKREMOTEINFO
{
	CString szRemoteIP; //远程主机IP
	UINT uRemotePort;   //远程主机端口号
	LPVOID pBuf;        //接收的数据
	int iLen;           //接收的数据长度
	LPVOID pParent;     //父指针
	HANDLE* phEvent;    //事件对象
};
