Microsoft Development Environment 扩展工具集

TRACE

VisualStudioNetExtendTools.Connect.CommentTRACE

功能：注释掉文件中的所有 TRACE  语句。

if(!AdoRecSet.Open(szSQL))
{
        /*TRACE(_T("%s(%d) : 向所有已登录用户发送信息,查找用户出错\n"), __FILE__,    __LINE__);*/
       return FALSE;
}

VisualStudioNetExtendTools.Connect.UncommentTRACE

功能：与上面命令相反，执行上面命令的逆操作。
