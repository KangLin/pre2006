康林

  1. 解决方案对象：Solution 对象   DTE.Solution
  2. 项目集合：Projects 集合       DTE.Solution.Projects 或 DTE.ActiveSolutionProjects()
  3. 项目对象：Project 对象        DTE.Solution.Projects.Item(n)
  4. 项目项集合：ProjectItems 集合 DTE.Solution.Projects.Item(n).ProjectItems
  5. 项目项对象：ProjectItem 对象  DTE.Solution.Projects.Item(n).ProjectItems.Item(m)
  6. 文档集合：Documents           DTE.Documents
  7. 文档对象：Document            DTE.ActiveDocument 或 DTE.Documents.Item(n)
  8. 文本文档对象：TextDocument

  1. 从文档（Document）得到项目项对象：由文档的属性 ProjectItem 属性（常规扩展性）得到。

  2. 从项目项得到所属的工程：由项目项的属性 ContainingProject 属性（常规扩展性）得到。
    例子：
    Dim doc as Document
    Dim pji as ProjectItem
    Dim prj as Project
    doc = DTE.ActiveDocument      '得到当前文档
    pji = doc.ProjectItem         '得到项目项
    prj = pji.ContainingProject   '得到工程
