如何在自己添加的对话框显示窗口图标?

1、在对话框类头文件中定义变量：
HICON m_hIcon;
2、在实现文件中的对话框构造函数中初始化变量：
m_hIcon = AfxGetApp()->LoadIcon(IDC_MAINFRAME);
3、在 OnInitDialog()
// 设置此对话框的图标。当应用程序主窗口不是对话框时，框架将自动
// 执行此操作
SetIcon(m_hIcon, TRUE); // 设置大图标
SetIcon(m_hIcon, FALSE); // 设置小图标

如何让对话框启动时最大化或最小化？

在 OnInitDialog()
ShowWindo(SW_SHOWMAXIMIZED); //最大化
Showindow(SW_SHOWMINIMIZED); //最小化
