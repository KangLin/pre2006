capSetCallbackOnVideoStream

康  林

你能用这个宏或者明确调用WM_CAP_SET_CALLBACK_VIDEOSTREAM 消息，设置一个应用程序的回调函数。在捕获期间，当视频缓冲区满后，AVICap 调用这个程序。

BOOL capSetCallbackOnVideoStream(
  hwnd,  
  fpProc 
);

参数：

hwnd
捕获窗口的句柄。
fpProc
视频流回调函数地址。这个参数指定为 NULL 时，卸载前面安装的视频流回调函数。

返回值：

如果成功返回 TRUE 如果正在捕获流或者单一帧捕获会话正在执行时，返回FALSE。（ FALSE if streaming capture or a single-frame capture session is in progress.）

注释：

捕获窗口在捕获帧写入硬盘前先调用这个回调函数。如果想修改帧的话，它允许应用程序修改帧。

如果一个视频流回调函数被用于视频捕获的话，那么这个函数要在捕获会话开始前安装，并且它应在会话期间保持激活。它能在捕获流结束后终止。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
