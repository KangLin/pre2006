capCaptureAbort

康  林

你能用此宏或者明确发送 WM_CAP_ABORT 消息，放弃捕获操作。

BOOL capCaptureAbort(
  hwnd  
);

参数：

hwnd
捕获窗口的句柄。

返回值：

成功返回 TRUE，否则，返回 FALSE。

注释：

捕获操作必须用此宏停止。

在捕获帧步中，在这个消息被发送前，收集的图象数据被保存在捕获文件中。但是音频不被捕获。（这与 capCaptureStop 不同）。

用 capCaptureStop 宏在当前位置挂起捕获步，然后捕获音频。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
