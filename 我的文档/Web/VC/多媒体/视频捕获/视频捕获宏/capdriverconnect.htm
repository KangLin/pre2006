capDriverConnect

康  林

你能用此宏或者明确发送 WM_CAP_DRIVER_CONNECT 消息，把捕获窗口联接到捕获驱动。

BOOL capDriverConnect(
  hwnd,  
  iIndex 
);
参数：
　
iIndex
捕获驱动的索引值。范围[0，9]。
hwnd
捕获窗口句柄。

返回值：

如果成功返回 TRUE，如果指定的捕获驱动不能被除捕获窗口联接，返回FALSE。

注释：

当捕获驱动与捕获窗口联接时，捕获窗口自动断开与前面的捕获驱动的联接。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
