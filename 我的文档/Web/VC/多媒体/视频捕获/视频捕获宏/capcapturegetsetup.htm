capCaptureGetSetup

康  林

你能用此宏或者明确发送 WM_CAP_GET_SEQUENCE_SETUP 消息，得到当前捕获流的参数设置。

BOOL capCaptureGetSetup(
  hwnd,  
  s,     
  wSize  
);

参数：

hwnd
捕获窗口的句柄。
s
 CAPTUREPARMS 结构地址。
　
wSize
s 结构的大小，单位：字节。

返回值：

成功返回TRUE，否则返回FALSE。

注释：

参数信息将用于控制捕获流。看 CAPTUREPARMS 结构。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
