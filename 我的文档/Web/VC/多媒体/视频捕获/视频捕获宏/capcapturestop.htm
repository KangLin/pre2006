capCaptureStop

康  林

你能用此宏或者是明确地发送 WM_CAP_STOP 消息，停止捕获操作。

在捕获帧步中，在这个消息被发送前，收集的图象数据被保存在捕获文件中。同样，如果允许一个音频捕获操作的话，收集的音频数据将被保存在捕获文件中。（这与 capCaptureAbort 不同）

BOOL capCaptureStop(
  hwnd  
);

参数：

hwnd
捕获窗口的句柄。

返回值：

成功返回 TRUE，否则返回 FALSE。

注释：

这个捕获操作必须用这个宏停止。用 capCaptureAbort 宏放弃当前捕获操作。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
