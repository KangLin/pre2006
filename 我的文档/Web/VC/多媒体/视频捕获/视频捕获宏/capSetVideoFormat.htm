capSetVideoFormat

康  林

你能用此宏或者明确地发送 WM_CAP_SET_VIDEOFORMAT 消息，设置捕获视频数据的格式。

BOOL capSetVideoFormat(
  hwnd,           
  psVideoFormat,  
  wSize           
);

参数：

hwnd
捕获窗口的句柄。
psVideoFormat
BITMAPINFO 指针。
wSize
psVideoFormat 指定结构的大小。单位：字节。

返回值：

成功返回 TRUE，否则返回 FALSE。

注释：

因为视频格式是设备的细节，所以应用程序应该检查它的返回值，以确定设备驱动是否接受此格式。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
