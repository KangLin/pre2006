capCaptureSetSetup

康  林

你能用此宏或者明确调用 WM_CAP_SET_SEQUENCE_SETUP 消息，用于设置捕获流。

BOOL capCaptureSetSetup(
  hwnd,        
  psCapParms,  
  wSize        
);

参数：

hwnd
捕获窗口句柄。
psCapParms
 CAPTUREPARMS 结构地址。
wSize
psCapParms 指定结构的大小，单位：字节。

返回值：

成功返回TRUE，否则返回FALSE。

注释：

用于设置捕获流，见 CAPTUREPARMS 结构。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
