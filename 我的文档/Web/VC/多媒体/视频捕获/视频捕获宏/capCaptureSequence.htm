capCaptureSequence

康  林

你能用这个宏或者明确发送 WM_CAP_SEQUENCE 消息到AVICap视窗口，开始捕获视频流和音频流到一个文件。

BOOL capCaptureSequence(
  hwnd  
);

参数：

hwnd
捕获窗口句柄。

返回值：

如果成功返回 TRUE，否则，返回 FALSE。

如果一个错误出现并且已经用 capSetCallbackOnError 宏设置了错误回调函数，那么这个错误回调函数被调用。

注释：

如果你想改变参数控制捕获流，那么在捕获开始前用 capCaptureSetSetup 设置。

在默认情况下在一个捕获窗口正在捕获运行期间，不允许其它捕获应用程序继续运行。要改就这个，要么设置结构  CAPTUREPARMS 中的成员函数 fYield 的值为 TRUE，或者安装一个产生（yield)回调函数。

在捕获流期间，捕获窗口能明确地向你的应用程序传送指明类型的（specific types of conditions）通知。要为这些通知安装回调函数，用下面的宏：

capSetCallbackOnError
capSetCallbackOnStatus
capSetCallbackOnVideoStream
capSetCallbackOnWaveStream
capSetCallbackOnYield

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
