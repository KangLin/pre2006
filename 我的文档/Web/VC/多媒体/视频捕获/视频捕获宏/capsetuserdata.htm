capSetUserData

康  林

你能用这个宏或者明确地调用 WM_CAP_SET_USER_DATA 消息，把一个 LONG 型的数据值联系到一个捕获窗口。

BOOL capSetUserData(
  hwnd,  
  lUser  
);

参数：

hwnd
捕获窗口的句柄。
lUser
与捕获窗口相联第的值。

返回值：

如果成功返回 TRUE，如果捕获正在进行，返回 FALSE。

注释：

典型地，这个消息用于为捕获窗口相联系的数据块的指针。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
