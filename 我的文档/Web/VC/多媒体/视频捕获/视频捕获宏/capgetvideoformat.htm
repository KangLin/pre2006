capGetVideoFormat

康  林

你能用此宏或者明确的发送 WM_CAP_GET_VIDEOFORMAT 消息。

DWORD capGetVideoFormat(
  hwnd,           
  psVideoFormat,  
  wSize           
);
hwnd
捕获窗口的句柄.
psVideoFormat
BITMAPINFO 结构的地址。你能用 NULL 得到 BITMAPINFO 所要的字节数。
wSize
     psVideoFormat 它所要的大小，单位：字节。

返回值：

返回大小，单位：字节。如果捕获窗口没有连接捕获驱动，则返回 0。视频格式要求一个调色板，当前调色板总是被返回。

注释：

因为视频格式的请求大小因压缩而改变，所以应用程序应该先请求分配一块本地内存。然后再请求得到视频格式数据。

Because compressed video formats vary in size requirements applications must first retrieve the size, then allocate memory, and finally request the video format data.

QuickInfo

  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
