capGetDriverDescription

康  林

得到捕获驱动的版本描述。

BOOL VFWAPI capGetDriverDescription(
  WORD wDriverIndex,  
  LPSTR lpszName,     
  INT cbName,         
  LPSTR lpszVer,      
  INT cbVer           
);

参数：

wDriverIndex
捕获驱动的索引值。范围[0，9]。

先枚举即插即用捕获驱动，接下来在注册表捕获驱动列表中查找，再在 SYSTEM.INI捕获列表中查找。

lpszName
包含相应捕获驱动名的字符串指针
cbName
由 lpszName 指定的缓冲区的长度，单位：字节。
lpszVer
包含捕获驱动相应的描述的字符串指针。
cbVer
由 lpszVer 指定的缓冲区的长度，单位：字节。

返回值：

成功返回 TRUE，否则，返回FALSE。

注释：

如果描述长度大于给定的缓冲区的长度，那么这个描述将被缩短。返回字符串总是以空终结符结束。如果缓冲区为 0，那么不复制相应的描述。

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
  Import Library: Use vfw32.lib.
  Unicode: Implemented as Unicode and ANSI versions on Windows NT.
