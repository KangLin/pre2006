capCreateCaptureWindow

康  林

建立一个捕获窗口。

HWND VFWAPI capCreateCaptureWindow(
  LPCSTR lpszWindowName,  
  DWORD dwStyle,          
  int x,                  
  int y,                  
  int nWidth,             
  int nHeight,            
  HWND hWnd,              
  int nID                 
);

参数：

lpszWindowName
包含捕获窗口名字的字符串指针。
dwStyle
捕获窗口的窗口类型。窗口类型的描述见 CreateWindowEx 函数.
x and y
捕获窗口的左上角的 x 和 y 坐标。
nWidth and nHeight
捕获窗口的宽度和高度。
hWnd
捕获窗口父窗口的句柄。
nID
窗口的标志符。

返回值：

成功返回捕获窗口的句柄，否则返回 NULL。

QuickInfo

  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.

  Import Library: Use vfw32.lib.
