waveInReset

康  林

停止指定的输入设备输入，并重新设置当前位置为0。标识所有的未用缓存为已用，即设置缓存 WAVEHDR 结构中的成员 dwFlags 为 WAVE_DONE.并返回到应用程序。

MMRESULT waveInReset(
  HWAVEIN hwi  
);

参数：

hwi
输入设备的句柄

返回值：

成功返回 MMSYSERR_NOERROR ，失败返回一个错误值：

Value Description
MMSYSERR_INVALHANDLE 指定的设备句柄无效
MMSYSERR_NODRIVER 无当前指定的设备
MMSYSERR_NOMEM 内在锁定或不能分配
