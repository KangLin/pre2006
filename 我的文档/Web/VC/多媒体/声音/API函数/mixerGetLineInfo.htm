mixerGetLineInfo

康  林

返回指定混频设备的信息。

MMRESULT mixerGetLineInfo(
  HMIXEROBJ hmxobj,  
  LPMIXERLINE pmxl,  
  DWORD fdwInfo      
);

参数：

hmxobj
控制指定音频线的混频器对象的句柄。
pmxl
 MIXERLINE 结构的地址，这个结构将背混频器驱动程序相关的音频线的信息填充。成员 cbStruct 必须用这个结构的大小初始化，单位：字节。
fdwInfo
关于返回音频线信息的标识。
MIXER_GETLINEINFOF_COMPONENTTYPE
参数 pmxl 返回关于结构 MIXERLINE 中成员 dwComponentType 中指定类型的音频线的第一个音频线的信息。这个标识用于返回一个指定的组合类型的音频线的信息。
MIXER_GETLINEINFOF_DESTINATION
参数 pmxl 返回关于在结构 MIXERLINE 中成员 dwDestination 中指定的目的音频线的信息。这个索引范围是从0到结构MIXERCAPS 中成员 dwDestinations 减一。All remaining structure members except cbStruct require no further initialization.
MIXER_GETLINEINFOF_LINEID
参数 pmxl 返回关于 MIXERLINE 结构中成员 dwLineID 指定的音频线的信息。 This is usually used to retrieve updated information about the state of an audio line. All remaining structure members except cbStruct require no further initialization.
MIXER_GETLINEINFOF_SOURCE
参数 pmxl 返回关于由 MIXERLINE 中成员 dwDestination 和 dwSource 指定的源音频线。The index specified by dwDestination ranges from zero to one less than the value in the cDestinations member of the MIXERCAPS structure. The index specified by dwSource ranges from zero to one less than the value in the cConnections member of the MIXERLINE structure returned for the audio line stored in the dwDestination member. All remaining structure members except cbStruct require no further initialization.
MIXER_GETLINEINFOF_TARGETTYPE
参数 pmxl 将返回关于 MIXERLINE 结构中 Target 结构中成员 dwType 指定的音频线的信息。这个标识用于返回关于一个音频线的对象类型的句柄的信息。（例如：MIXERLINE_TARGETTYPE_WAVEOUT）。当用这个标识调用函数 mixerGetLineInfo 前，MIXERLINE 结构的成员 dwType,wMid,wPid,vDriverVersion 和 szPname。All of these values can be retrieved from the device capabilities structures for all media devices. Remaining structure members except cbStruct require no further initialization.
MIXER_OBJECTF_AUX
参数 hmxobj 是一个辅助设备标识符，范围从0到 auxGetNumDevs 减一。
MIXER_OBJECTF_HMIDIIN
参数 hmxobj 是一个设备 MIDI 输入设备的句柄。这个句柄必须是一个由函数 midiInOpen 返回的句柄。
MIXER_OBJECTF_HMIDIOUT
参数 hmxobj 是一个设备 MIDI 输出设备的句柄。这个句柄必须是由函数 midiOutOpen 返回的。
MIXER_OBJECTF_HMIXER
参数 hmxobj 是一个混频设备的句柄，这个句柄是这个函数 mixerOpen 返回的。这个标识是可选的。
MIXER_OBJECTF_HWAVEIN
参数 hmxobj 是一个波形音频输入设备句柄，这个句柄是由函数 waveInOpen 返回的。
MIXER_OBJECTF_HWAVEOUT
参数 hmxobj 是一个波形音频输出设备的句柄，这个句柄是由函数 waveOutOpen 返回的。
MIXER_OBJECTF_MIDIIN
参数 hmxobj 是一个 MIDI 输入设备标识符。这个标识符必须是一个范围从0到函数 midiInGetNumDevs 减一。
MIXER_OBJECTF_MIDIOUT
参数 hmxobj 是一个 MIDI 输出设备标识符。这个标识符必须是一个范围从0到函数 midiOutGetNumDevs 减一。
MIXER_OBJECTF_MIXER
参数 hmxobj 是一个混频设备的标识符，范围从0到函数 mixerGetNumDevs 减一。这个标识是可选的。
MIXER_OBJECTF_WAVEIN
参数 hmxobj 是波形音频输入设备的标识符。范围从0到函数 waveInGetNumDevs 减一。
MIXER_OBJECTF_WAVEOUT
参数 hmxobj 是波形音频输出设备的标识符。范围从0到函数 waveOutGetNumDevs  减一。

返回值：

如果成功返回 MMSYSERR_NOERROR ，否则返回一个错误。

Value Description
MIXERR_INVALLINE The audio line reference is invalid.
MMSYSERR_BADDEVICEID The hmxobj parameter specifies an invalid device identifier.
MMSYSERR_INVALFLAG One or more flags are invalid.
MMSYSERR_INVALHANDLE The hmxobj parameter specifies an invalid handle.
MMSYSERR_INVALPARAM One or more parameters are invalid.
MMSYSERR_NODRIVER No mixer device is available for the object specified by hmxobj.
