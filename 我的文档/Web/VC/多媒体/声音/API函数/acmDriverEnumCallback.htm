acmDriverEnumCallback

康  林

这个回调函数被用于函数 acmDriverEnum 。

BOOL ACMDRIVERENUMCB acmDriverEnumCallback(
  HACMDRIVERID hadid,  
  DWORD dwInstance,    
  DWORD fdwSupport     
);

参数：

hadid
一个 ACM 驱动程序的标识符句柄。
dwInstance
由应用程序通过 acmDriverEnum. 传给回调函数的值。
fdwSupport
由参数 hadid 指定的设备驱动程序所支持设备的细节标识。这个标识与结构 ACMDRIVERDETAILS 中的成员 fdwSupport 一样。这个参数能包含下列的值：
ACMDRIVERDETAILS_SUPPORTF_ASYNC
驱动程序支持异步转换。
ACMDRIVERDETAILS_SUPPORTF_CODEC
驱动程序支持在两种不同格式名称的两种不同的格式之间的转换。例如：如果这个驱动程序支持从 WAVE_FORMAT_PCMD 到 WAVE_FORMAT_ADPCM 的压缩，那么这个标识被设置。
ACMDRIVERDETAILS_SUPPORTF_CONVERTER
驱动程序支持在同种格式名称的两种不同的格式之间的转换。例如：如果驱动程序支持 WAVE_FORMAT_PCM 重取样。这个标识被设置。
ACMDRIVERDETAILS_SUPPORTF_DISABLED
驱动程序是无效的。应用程序必须用函数 acmDriverEnum 向回调函数指定 ACM_DRIVERENUMF_DISABLED ，标识驱动程序失效。
ACMDRIVERDETAILS_SUPPORTF_FILTER
驱动程序支持过滤器（指不改变格式的任何属性而修改数据）。例如：如果一个驱动程序支持在 WAVE_FORMAT_PCM 音量或回声操作，这个标识被设置。

返回值：

这个函数继续相枚举返回 TRUE，要停止枚举返回 FALSE。

注释：

如果没有 ACM 设备被安装，函数 acmDriverEnum 将返回 MMSYSERR_NOERROR (zero) ，这个个回调函数不会被调用。

下列函数不能在回调函数中被除调用：acmDriverAdd, acmDriverRemove, 和 acmDriverPriority.
